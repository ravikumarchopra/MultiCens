About

MultiCens is platform from where we can predict hormone-responsive genes in a target tissue by providing the inputs such as Source and target tissue names, the mediator hormone, and the list of genes from source and the target tissues. The tool will take the list of genes from the source tissue and will rank the hormone-responsive genes from the target tissue gene list by their responsiveness for the selected hormone. And we can also measure the Centrality scores such as local centrality, global centrality, and query-set centrality.

  1. Local Centrality
    A node in a layer (i.e. Tissue) can affect other nodes (i.e. genes) in the same layer as well as different layers. Local centrality captures the within-layer effect of a node.
  2. Global Centrality
    Local centrality considers the effect of only within-layer connections, Global centrality captures the remaining effect. The global centrality of a node is a measure of its influence on all nodes irrespective of their layers.
  3. Query Set Centrality
    Query-set centrality captures the effect of a node on a query-set of nodes present in any specific layer in the multilayer network.


Tool v1.0

Switch to the Tool v1.0 tab to use the tool, provide the required inputs, by selecting the source and target tissue name, and hormone name from the dropdowns and you can download the sample csv files from the bottom-right corner of the Tool tab and then browse them from the browse option for source and target gene list. Finally, click the Get Ranking button to get the ranking. It will take a few (3-5) minutes to produce the results as it using very large correlation matrices in the backend. How to use
  • Step 1: Add tissues to the Tissue List to create the network by following the following steps:
    1. Select or type Tissue Name.
    2. Select dataset (default:GTex) from Select Dataset dropdown. If you want to use your own gene expression data, then select Others option.
    3. If you selected Others option from Dataset dropdown, Then select a compressed (.zip, .gz) csv file.
  • Step 2: Select the Centrality Measure from Select Centrality Measure dropdown.
  • Step 3: Click the Get Ranking button.
  • Step 4: Please wait for 3-5 min(s).
  • Step 5: Scroll down to see the results.
  • Step 6: You can filter the results using Search option.

Tool v1.0

Form fields
  1. Source Tissue
    The source tissue for which we are selecting CSV file with gene list in the 4th field (CSV file field for source gene list) and want to predict hormone-responsive genes in a target tissue.
  2. Target Tissue
    The target tissue for which we are selecting CSV file with a gene list in the 5th field (CSV file field for target gene list) and want to predict hormone-responsive genes inside.
  3. Hormone
    The mediating hormone between the source and the target tissue.
  4. CSV file with source gene list
    The list of genes from the source tissue for which we want to predict selected hormone-responsive genes in the target tissue.
  5. CSV file with target gene list
    The list of genes from the target tissue for which we want to predict selected hormone-responsive gene ranking.

Tool v2.0 (beta)

Form fields
  1. Tissue List
    The Tissue List contains the list of tissues we are using to build the network.
  2. Tissue Name
    Name of the Tissue.
  3. Dataset
    Dataset Name (by default, it will use GTex dataset). If you want to use your own gene expression data, then select Others option.
  4. Gene Expression Data File
    A compressed (.zip, .gz) csv file.
  5. Centrality Measure
    There are the following three Centrality Measures which we can use to measure the centrality of genes using the network we are creating for tissues from the Tissue List.
    1. Local Centrality
      A node in a layer (i.e. Tissue) can affect other nodes (i.e. genes) in the same layer as well as different layers. Local centrality captures the within-layer effect of a node.
    2. Global Centrality
      Local centrality considers the effect of only within-layer connections, Global centrality captures the remaining effect. The global centrality of a node is a measure of its influence on all nodes irrespective of their layers.
    3. Query Set Centrality
      Query-set centrality captures the effect of a node on a query-set of nodes present in any specific layer in the multilayer network.
  6. Gene Query Set
    Gene query set is a csv file containing a list of genes for the query set that we will be using to measure the query set centrality.
